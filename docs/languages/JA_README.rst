                     OpenHotspot Framework 0.1.4
                     ===========================

OpenHotspotは、機械学習、犯罪分析のフレームワークで書かれた C ++ 11およびパターン認識、
ホットスポットの速度とスケーラビリティのために構築されています 検出と位置に基づいて、犯罪を予測する。
